# 更新日志

## 0.4.0 - 2026-05-19

### 新增

- 设置页改为模块管理台：侧边栏只保留模块管理、访问密钥、导入导出。
- 模块管理新增官方模块、自定义完整模块、补充拓展包、外观模块分组。
- 模块卡片新增启用开关、官方/自定义/拓展/外观标签、说明文本和设置入口。
- 单个模块设置改为独立详情页，并在左上角提供返回模块管理按钮。
- 新增外观模块分类，区分全局替换与补充拓展；多个全局替换同时启用时显示红色冲突提示。
- 人物印象模块新增日记后自动识别、写入程度、更新策略、是否允许新建候选档、最低置信度等设置。

### 修复

- 定时归档任务改为后台 Agent 工具循环执行，任务提示词不再作为可见消息发到聊天窗口。
- 归档后的可见提示只由“归档后是否提示”控制，不再泄漏执行提示词。
- 日记写入后不再因为 `people` 字段出现新称呼就自动建档人物印象。
- 写日记工具返回内容不再附带人物印象自检提示词。

### 文档

- 更新模块化目录、人物印象策略、外观模块、导入导出和隐藏归档提示词说明。
- 更新官方模块说明文件，明确人物印象与 WebUI 模块的职责边界。
